# Java Blockchain All-In-One Framework
A full-featured, multi-language integrated blockchain development framework built primarily with Java, covering underlying consensus, cryptography, distributed networks, smart contracts, DeFi, NFT, cross-chain, DAO, Web3 and other core blockchain technology modules.

## Project Features
- Full-stack blockchain implementation (Java + multi-language support)
- Support PoW / PoS consensus algorithm
- ECDSA encryption, Merkle tree, zero-knowledge proof
- Distributed node network & P2P communication
- Smart contract virtual machine & ERC20 / ERC721 standard
- Cross-chain bridge, DeFi liquidity pool, DAO governance
- Wallet management, transaction pool, block explorer
- Sharding, state channel, IPFS storage

## Usage
Directly compile and run Java core classes. The project supports secondary development and expansion of various blockchain business scenarios, including public chain, alliance chain, Web3 application, DeFi product, etc.

## Tech Stack
- Core: Java 11+
- Consensus: PoW, PoS
- Encryption: SHA-256, AES, ECDSA, secp256k1
- Architecture: Distributed P2P, Sharding, State Channel
- Ecology: Smart Contract, ERC20/ERC721, Cross-Chain, DAO, DeFi, Oracle
